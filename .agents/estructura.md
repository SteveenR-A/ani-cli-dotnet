# Estructura del Proyecto (Structure)

Este documento describe la arquitectura y estructura de directorios del proyecto `ani-cli-dotnet`.

- `/Installer`: Contiene el código de configuración de WiX y el instalador.
- `/Terminal`: Lógica para interactuar con la terminal (colores, selección de menús).
- `/Extractors`: Clases y lógicas de extracción de datos (web scraping de anime).
- `/Tests`: Scripts y datos de prueba.
- `/Models`: Entidades y modelos de datos.
